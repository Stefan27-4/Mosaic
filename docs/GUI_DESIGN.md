## Color Scheme

**Background:**     #0F172A (Deep Midnight Blue)
**Cards:**          #1E293B (Dark Slate)
**Accent/Buttons:** #8B5CF6 (Vivid Purple)
**Text:**           #F8FAFC (Off-white)
**Secondary:**      #94A3B8 (Slate Gray)

**Budget Bar Colors:**
- Green (#10B981): 0-49% usage
- Yellow (#F59E0B): 50-79% usage
- Red (#EF4444): 80-100% usage

**Debug Log:**
- Background: #0A0F1E (Very dark blue-black)
- Text: #22C55E (Matrix green)
- Border: #22C55E (Matrix green, 2px)

## Typography

- **Headers:** Segoe UI, 24-32pt, Bold
- **Body:** Segoe UI, 12-14pt, Regular
- **Buttons:** Segoe UI, 14-16pt, Bold
- **Debug Log:** Consolas, 10pt, Regular (monospace)

## Layout Details

### SetupView
- Window: 600x700px, centered
- Padding: 30px all sides
- Card corner radius: 15px
- Input fields: 35-40px height
- Button: 50px height, full width
- All inputs have 2px purple border when focused

### MainChatView
- Window: 1200x800px
- Sidebar: 300px width (25%)
- Chat area: Remaining width (75%)
- Padding: 20px outer, 15px inner
- Message cards: 10px corner radius
- Progress bar: 20px height, 10px corner radius
- Input field: 80px height
- Send button: 80px height, 100px width
- Debug log: 150px height when shown

## Interactive Elements

### Budget Health Dashboard
- Updates in real-time
- Progress bar animates smoothly
- Color transitions: green â†’ yellow â†’ red
- Text updates: "Spent: $X.XX / $Y.YY"
- Warning shown when > 80%
- Auto-halt when >= 100%

### Smart Router Toggle
- ON (default): Model selector disabled
- OFF: Model selector enabled
- System message on toggle
- Purple accent when ON

### Debug Log
- Hidden by default
- Checkbox to show/hide
- Smooth expand/collapse animation
- Auto-scrolls to bottom on new messages
- Timestamps in [HH:MM:SS] format

### Chat Messages
- User messages: Right side, dark slate background
- AI messages: Left side, purple background
- System messages: Center, gray italic text
- Auto-scroll to latest message
- Word wrap at 600px
